# StudyHelper

강의 내용을 과목별로 관리하고, 간단한 AI를 통해 분류하며학습자료를 기반으로 퀴즈를 만들 수 있는 웹서비스입니다.

## 프로젝트 소개

대학교에서 여러 전공과목을 수강하다 보면 강의내용, 시험 정리, 과제 메모가 여러 파일과 프로그램에 흩어지게 됩니다.

StudyHelper는 학습자료를 한곳에 모아 저장하고 검색과 퀴즈 기능을 통해 시험 공부에 다시 활용할 수 있도록 하는 프로젝트입니다.

## 주요 기능
- 회원가입 및 로그인
- 사용자별 과목 관리
- 학습자료 생성, 조회, 수정, 삭제
- 제목, 본문, 태그 검색
- 머신러닝을 이용한 과목 자동 추천
- 학습자료 기반 퀴즈 생성
- 퀴즈 풀이 및 채점
- 오답노트와 학습 통계
